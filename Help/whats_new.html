Launcher

What's new in this version?

  • Added ability to open SoftTest test cases on ISD QA site
  • Added ability to define which browser to use for which URL (tasks, defects, test cases, HF, SIF)
  • Added ability to define explicitly how to open URLs (new window or new tab) for browsers that support it: Mozilla Firefox, Google Chrome, Opera
  • Fixed bugs in command line interface features caused by changes in iTMS 2.1.0.3
  • Updated HTML help



History of changes in previous versions

Version 3.1

  • Added ability to open Software Installation Forms
  • Added ability to open Hotfixes in:
    • Short format (without revision specified) - HF 1.3590
      In this case, "View Task" button opens web-page with ACTIVE revisions of the hotfix.
      Other buttons "Children", "Parent" and "Related" open web-page with ALL revisions of the hotfix.
    • Full format (with revision specified) - HF 1.3590.8
      In this case, web-page with detailed information about specified hotfix revision opens.
  • Added "Restore" button with commands to restore SIF and HF records on Defects page
  • Allow saving characters with ASCII code > 127 in "Separators" and "Tasks Separators" fields
  • Added new default separators: No-Brake Space, En Dash, Em Dash. This will allow to open tasks with these separators
  • Several bugs fixed

Version 3.0

  • Tasks that have digits in client name like MAR2A-00045, DEV1-00001 can be opened now
  • Added support for MS Windows Vista and Internet Explorer 8
  • Added new "Related tasks" button to main window
  • Added "Other" page to Settings window with:
    • Ability to customize transparency of main window
    • Hotkey for fast window activation
    • Flag to set focus to "Task Name" field when window is activated
    • Flag to highlight text in "Task Name" field when window is activated
  • "Autologin" feature can be automatically enabled in IE after user enters "Login"/"Password" for an URL
  • Added ability to reassign already used hotkeys in URLs
  • Added ability to open and copy tasks from History page of Settings window
  • Small user interface changes
  • Command line syntax has been extended (see Command line interface for more details):
    • Added option that allows to view SPC linked tasks for the list of given ones:
      TMS_Launcher -spc -name "NAME" -prod "PROD" -stat "STAT" -msp "MSP" input_file output_file
    • Added option that allows to view PRODUCT, STATUS, MS PROJECT for the list of given tasks:
      TMS_Launcher -inf -name "NAME" -prod "PROD" -stat "STAT" -msp "MSP" input_file output_file
    • Added option that allows to find tasks that match or do not match the given regular expression
      TMS_Launcher -exp "EXPRESSION" input_file output_file
    • Negative search logic can be turned on by adding "!" before an expression in all filters
  • Several bugs fixed

Version 2.7

  • Command line syntax has been extended (see Command line interface for more details):
    • Added option that allows to view timesheets for the list of tasks
      TMS_Launcher.exe -t [-name "NAME"] input_file output_file
    • Added new parameter [-stat "STAT"] to filter child tasks by status
      TMS_Launcher.exe -c [-name "NAME"] [-prod "PROD"] [-stat "STAT"] input_file output_file
    • Added new parameter [-name "NAME"] to filter parent tasks by names
      TMS_Launcher.exe -p [-name "NAME"] input_file output_file
    • Now parameters -name, -prod, -stat are regular expressions, not simple masks
  • Several bugs fixed

Version 2.6

  • New predefined URL "SPC linked tasks" has been added to URLs page to open tasks in SPC system
  • New parameter %AA_ID% is now handled in URLs to open tasks in SPC system
  • Fixed bug when help file cannot be opened sometimes if it is called by <F1> from the program

Version 2.5

  • Getting QR actions in addition to QB ones for a list of tasks
  • Improved list controls usability in Settings window
  • Fixed bug with opening defects in SoftTest 1.5.1
  • Fixed bug with sorting of context menu items for View Task button
  • Updated HTML help

Version 2.4

  • Keeping the history of opened tasks in Windows Registry
  • History page was added to settings window to manage history
  • Possibility to enable/disable filling up Task ID with starting '0' to 5-character length was added to Format page
  • Fixed bug with incorrect reading of child tasks/defects for a list of given ones
  • Updated HTML help

Version 2.3

  • SoftTest and TMS passwords are encrypted before saving to Windows Registry
