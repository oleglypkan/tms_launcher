Launcher 2.1

What's new in this release?

  • Parent task for the given one can be opened
  • Parent and child defect(s) for the given one can be opened on ISD QA site
  • Ability to define different URLs for defects from different SoftTest projects
  • Ability to open defects in format "ST_LABGUI_SYNCH-12345", not just "LAB-12345"
  • Ability to choose "Task Name" field control: RichEdit or ComboBox with history of opened tasks
  • Command line interface support with:
    • Ability to get the list of parent tasks for a list of the given ones
    • Ability to get the list of child tasks for a list of the given ones
  • Improvements to product usability including redesigned options interface
  • Small bug fixes
