TMS Launcher 2.0

General page of settings window

General page of settings window allows to change the following settings:

Common section:

  • Minimize at startup
    Enable this flag to have main program window minimized at startup
  • Expand at startup
    Enable this flag to have main program window expanded at startup
  • Single click to hightlight
    When checked this option allows to highlight all the text in "Task Name" field of main window by single click.
  • Start with Windows
    Check this option to have the program started automatically during Windows startup

Select "View Task" or "View Child Tasks" option in "After right click in "Task Name" field" section to open task or its child tasks after you right click in "Task Name" field.
NOTE: task name is read from Windows Clipboard in this case, not from "Task Name" field.

Two options from the last section allow to:

  • Path to browser
    Choose browser that will be used to open tasks
  • Use system default browser to open single task
    Specify that system default browser should be used to open single task even if another browser is specified in "Path to browser" field. In other words, if the option is checked and you try to open the only task it will be opened in system default browser, while multiple tasks will be opened in browser defined in "Path to browser" field.
