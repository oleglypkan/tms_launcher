TMS Launcher 2.0

Format page of settings window

Format page allows to change format of task names and separators for tasks.
General format of task is the following: [%CLIENT%-]%ID%[-%EXT%].
Where %CLIENT% is client name of a task, %ID% is number of a task, %EXT% is an extension.
Use can specify the range length for each of these items and their separators.
"Tasks separators" allows to specify characters that can be used as tasks separators.
