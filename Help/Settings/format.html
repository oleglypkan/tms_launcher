Launcher 2.1

Format page of settings window

Format page allows to change format of task names and separators for tasks.

General format of task is the following:
[%CLIENT%-]%ID%[-%EXT%],

where %CLIENT% is client name of a task, %ID% is number of a task, %EXT% is an extension, "-" is separator that can be any of symbols defined in "Separators" field.

You can specify the range length for each of these items and their separators.
"Tasks separators" allows to specify characters that can be used as tasks separators. "History items" fields allows to specify maximum number of tasks that will be saved in history.
