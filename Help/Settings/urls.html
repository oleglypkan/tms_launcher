Launcher 2.1

URLs page of settings window

Available URLs section shows the list of URLs that can be used to open tasks. It allows to create, modify and remove URLs from the list.

For each URL it is possible to define:

  • URL caption
  • Link to open task
  • Link to open child tasks
  • Login, Password
  • 3 hotkeys to be used to open tasks, child and parent tasks correspondingly

NOTE: when you use a hotkey a task name is read from Windows Clipboard, not from "Task Name" field.

By default open tasks in section allows to choose which URL will be used to open task(s) when user uses buttons on main window and systray menu commands.

There is a useful feature that allows to enter login and password that will be passed to the browser when you open a site if required.
To use this feature you can enter your login and password to an URL in the following form:

http://login:password@url

If login and password are entered in URL in the format above they will be used instead of login and password entered in "Login" and "Password" fields.

Note that Microsoft Internet Explorer does not allow to use this feature by default. It is necessary to change some settings in Window Registry to allow IE to use this feature. Necessary reg file can be found HERE.
