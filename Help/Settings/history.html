History page of settings window

History page allows to manage the list of previously opened tasks.
"View" button allows to open selected task(s) in browser using default URL.
"Copy" button allows to copy selected task(s) to Windows Clipboard.
"Delete" and "Clear" buttons allow to delete selected task(s) from the list and clear the whole list correspondingly.
"Max items" field allows to specify maximum number of tasks that can be stored in history list.
