SoftTest page of settings window

SoftTest page allows to specify some SoftTest settings:

  • Path to SoftTest
    Full path to SoftTest binary file including file name
  • SoftTest filter name
    Name of the filter that will be used by the program to open defects. The name should be unique for each SoftTest project. So, %PROJECT% variable should not be removed from the filter name.
  • Login
    User's login name to enter to SoftTest application
  • Password
    User's password to enter to SoftTest application

NOTE: Both Login and Password are stored in Windows Registry in not in cipher form. So, if someone has administrator privileges on your PC he can view them.
