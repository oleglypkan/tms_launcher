TMS Launcher 2.0

Defects page of settings window

Defects page allows to define a set of records that will be used to decide if a task is TMS task or SoftTest defect. Each record contains a pair of strings: Client and Project. Client name of entered task is compared with Client string of all records and if match is found the task is considered as defect and opened by URL defined in "URL to open defects" field.
