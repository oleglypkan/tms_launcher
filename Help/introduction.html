Launcher 2.1

Introduction

TMS Launcher is a tool that allows user to open TMS tasks, SoftTest defects and links in a browser window. The tool does not require installation (only one executable file), it is easy to configure and use.

System requirements: IBM PC with one of the following OS: Windows 95/98/Me, Windows NT 4.0/2000/XP/2003. A WEB browser (Microsoft Internet Explorer, Opera, Netscape Navigator, etc) should be also installed in the OS.
