Launcher

Introduction

TMS Launcher is a tool that allows user to open TMS tasks, SoftTest defects and links in a browser window. Also since version 2.2 defects can be opened in SoftTest application directly. The tool does not require installation (only one executable file), it is easy to configure and use.

System requirements: IBM PC with one of the following OS: Windows 95/98/Me, Windows NT 4.0/2000/XP/2003. A WEB browser (Microsoft Internet Explorer, Opera, Netscape Navigator, Mozilla FireFox, etc) should be installed in the system. To open defects in SoftTest application it should be also installed.
