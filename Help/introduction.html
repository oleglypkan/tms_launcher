Launcher

Introduction

TMS Launcher is a tool that allows user to open TMS tasks, SoftTest defects, Test Cases, Software Installation Forms, Hotfixes and links in a browser window. Also since version 2.2 defects can be opened in SoftTest application directly. The tool does not require installation (only one executable file), it is easy to configure and use.

System requirements: IBM PC with one of the following OS: Windows 98/Me/2000/XP/2003/Vista/2008/7. A WEB browser (Microsoft Internet Explorer, Mozilla FireFox, Google Chrome, Opera, etc) should be installed in the system. SoftTest application should also be installed to open defects in it.
