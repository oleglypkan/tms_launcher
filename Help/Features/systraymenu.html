System tray menu

  System tray menu allows user to perform the following actions:

  • Show/hide main window
  • View task, child, parent and related tasks entered to "Task Name" field as well as located in Windows Clipboard
  • Open any page of TMS Launcher settings window
  • Clear history of processed tasks in combobox control
  • Close the program

NOTE: if you click on "x" button in the main window the window will just hide, the program will not close. To actually close the program please use system tray menu.
