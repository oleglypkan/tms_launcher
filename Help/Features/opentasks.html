Opening TMS tasks

Main window with RichEdit control as "Task Name" field Main window with ComboBox control as "Task Name" field

TMS Launcher allows to open TMS tasks in several ways:

  • by entering task name in main window and clicking the button
  • by using context menu of main window buttons
  • by clicking the right mouse button in the field of main window
  • by selecting systray menu item
  • by pressing predefined hotkey

The program is able to open specified task, its parent or child task(s).
It is possible to open single task as well as multiple tasks at one time. In case of multiple tasks opening they should be separated by characters specified in Format page of settings window.
The program can use task name entered in its main window or read it from Windows Clipboard (in case of using right mouse button click and hotkey).
There is a possiblity to define several sets of URLs that are used to open tasks, assign system wide hotkeys to open tasks by specific URLs.
It is possible to define the tasks format that can be used to validate tasks names for correctness.
It is also possible to select specific WEB browser to be used to open tasks or use system default browser.
In order the program is able to open parent task for the given one, it is necessary to specify Login and Password for URL that is selected as default in URLs page of settings window.
All the settings are accessible from TMS Launcher settings window.

Each successfully opened task is added to the history list and can be opened later again. History is stored in Windows Registry and is not removed when program is closed. History page of settings window provides ability to manage the history and specify maximum number of history items.

There is ability to choose "Task Name" field control: RichEdit or ComboBox.
Viewing history of previously opened tasks and opening them is possible in case of using ComboBox control only.

There is a difference in right-click action in RichEdit and ComboBox controls.
You can choose one of the following actions to be performed after you right-click in RichEdit control:

  • View Task
  • View Child Tasks
  • View Parent Task
  • View Related Tasks
  • Open context menu

Right-click in edit box of combo box control always opens context menu. However, right-click on button with down arrow will cause one of the following actions:

  • View Task
  • View Child Tasks
  • View Parent Task
  • View Related Tasks
