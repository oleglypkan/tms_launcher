TMS Launcher 2.0

Opening TMS tasks

TMS Launcher allows to open TMS tasks in several ways:

  • by entering task name in main window and clicking the button
  • by clicking the right mouse button in the field of main window
  • by selecting systray menu item
  • by pressing predefined hotkey
The program is able to open specified task or its child tasks.
It is possible to open single task as well as multiple tasks at one time. In case of multiple tasks opening they should be separated by characters specified in Format page of settings window.
The program can use task name entered in its main window or read it from Windows Clipboard (in case of using right mouse button click and hotkey).
There is a possiblity to define several sets of URLs that are used to open tasks, assign system wide hotkeys to open tasks by specific URLs.
It is possible to define the tasks format that can be used to validate tasks names for correctness.
It is also possible to select specific WEB browser to be used to open tasks or use system default browser.
All the settings are accessible from TMS Launcher settings window.
