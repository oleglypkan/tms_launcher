Launcher

Command line interface

  Since version 2.1 command line interface is supported. In later versions the syntax was extended and in current version is the following:

TMS_Launcher.exe -p [-name "NAME"] input_file output_file
NAME - task name mask. It is regular expression found parent task name should match to be displayed. -name is optional parameter
input_file - file that contains the list of tasks/defects
output_file - file that will contain the list of parent tasks/defects for the given ones
TMS_Launcher.exe -c [-name "NAME"] [-prod "PROD"] [-stat "STAT"] input_file output_file
NAME - task name mask. It is regular expression found child task name should match to be displayed
PROD - product version mask. It is regular expression product version should match to be displayed
STAT - task status mask. It is regular expression task status should match to be displayed
input_file - file that contains the list of tasks/defects
output_file - file that will contain the list of child tasks/defects for the given ones
Parameters -name, -prod and -stat are optional. If they are not specified all child tasks for the given parent will be displayed
TMS_Launcher.exe -spc [-name "NAME"] [-prod "PROD"] [-stat "STAT"] [-msp "MSP"] input_file output_file
NAME - task name mask. It is regular expression found SPC linked task name should match to be displayed
PROD - product version mask. It is regular expression product version should match to be displayed
STAT - task status mask. It is regular expression task status should match to be displayed
MSP - MS Project mask. It is regular expression for MS Project where task should be included to be displayed
input_file - file that contains the list of tasks
output_file - file that will contain the list of SPC linked tasks for the given ones
Parameters -name, -prod, -stat and -msp are optional. If they are not specified all SPC linked tasks for the given one will be displayed
TMS_Launcher.exe -qb input_file output_file
input_file - file that contains the list of tasks
output_file - file that will contain information about QB and QR actions (Action #, Date, Time, Person, Assigned To) for the given tasks
TMS_Launcher.exe -qc input_file output_file
input_file - file that contains the list of tasks
output_file - file that will contain information about QC actions (Action #, Date, Time, Person, Requirements #'s) for the given tasks
TMS_Launcher.exe -t [-name "NAME"] input_file output_file
NAME - SCC EMPLOYEE ID mask. It is regular expression SCC EMPLOYEE ID should match to be displayed. -name is optional parameter
input_file - file that contains the list of tasks
output_file - file that will contain the list of timesheets for the given tasks
TMS_Launcher.exe -inf [-name "NAME"] [-prod "PROD"] [-stat "STAT"] [-msp "MSP"] input_file output_file
NAME - task name mask. It is regular expression task name should match to be displayed
PROD - product version mask. It is regular expression product version should match to be displayed
STAT - task status mask. It is regular expression task status should match to be displayed
MSP - MS Project mask. It is regular expression for MS Project where task should be included to be displayed
input_file - file that contains the list of tasks
output_file - file that will contain information (PRODUCT, STATUS, MSP) about given tasks
Parameters -name, -prod, -stat and -msp are optional. If they are not specified all tasks will be displayed
TMS_Launcher.exe -exp "EXPRESSION" input_file output_file
input_file - file that contains the list of tasks
output_file - file that will contain the list of tasks whose content matches regular expression. All matched sub-expressions will also be printed

In order command line features work, it is necessary to specify Login and Password for URL that is selected as default in URLs page of settings window.
