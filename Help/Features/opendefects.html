Launcher 2.1

Opening SoftTest defects

SoftTest defects are considered by TMS Launcher as specific type of TMS tasks. TMS tasks with client names that match with predefined strings are supposed to be defects and opened by specified link.
There is a possibility to assign a SoftTest project from which defect will be read and URL to open defect for each such string.
Described settings can be made in Defects page of settings window.
